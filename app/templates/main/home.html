Selector

This page takes data in csv file format. Following algorithms are used on dataset to train models.

  • Decison Tree
  • K-Nearest Neighbour
  • Multilayer Perceptron
  • Random Forest
  • Support Vector Machines

After training and selection of most-suited model, model name is displayed to user.

Predictor

This page provides predictions on user provided datasets. This page contains two lists for selection purposes.

  • A list of datasets previously uploaded by the user.
  • The list contains the best models obtained from previous datasets.
Users are allowed to select one dataset and one model for testing, the application will provide a table of predictions with help of these two objects.

Performance

This page provides a user list of the best models obtained from previous datasets for selection. After selection of model the page will display performance metrics of model. The performance metrics used are:

  • Accuracy score
  • F1 score
  • Precision score
  • Recall score
  • Area under ROC
  • Prediction time

Cross-Performance

This page provides users with two lists.

  • List of models trained on previous datasets.
  • List of datasets which are used for training said datasets
The user is required to select one model from the model list and two datasets from the dataset list. The page will provide a comparative chart of performance metrics based on provided objects.
